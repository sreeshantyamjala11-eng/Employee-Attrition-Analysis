# Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition trends using Python and data visualization techniques to identify factors influencing workforce turnover.

The analysis focuses on employee demographics, salary, and years at company to discover patterns affecting employee retention.

---

# Objectives
- Identify key reasons for employee attrition
- Analyze employee demographics and salary trends
- Visualize attrition patterns using charts and graphs
- Generate business insights for HR decision-making

---

# Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# Key Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Univariate Analysis
- Bivariate Analysis
- Data Visualization

---

# Key Insights
- Younger employees showed higher attrition rates
- Employees with lower monthly income had increased turnover
- Employees with fewer years at company were more likely to leave

---

# Visualizations
The project includes:
- Histograms
- Pie Charts
- Bar Charts
- Attrition Trend Analysis

---

# Business Impact
The insights generated from this analysis can help HR teams improve employee retention strategies and workforce planning.
